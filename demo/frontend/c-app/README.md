# C端用户应用 - 简化版

## 功能说明

### 核心功能
1. **微信登录**（主要）- 扫码或微信内打开
2. **手机号验证**（备用）- Web端快速查看
3. **查看报告** - 展示健康报告
4. **下载PDF** - 一键下载报告

### 技术栈
- Vue 3 + Vite
- Vue Router
- Pinia (状态管理)
- Axios (API)

## 快速启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 访问地址
- 开发环境：http://localhost:5174
- 后端API：http://localhost:5000

## 页面结构

1. **首页** (`/`) - 落地页，引导用户登录
2. **验证页** (`/verify`) - 手机号验证（Web端）
3. **报告列表** (`/reports`) - 查看所有报告
4. **报告详情** (`/report/:id`) - 查看单个报告详情

## 与B端对接

B端管理员可以通过管理后台看到：
- C端用户（线索）
- 对话记录
- 生成的报告
- 下载统计
