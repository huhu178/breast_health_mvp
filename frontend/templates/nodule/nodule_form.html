💡 功能说明
  • 支持单结节或多结节组合管理
  • 智能表单：根据选择的结节类型动态显示相应字段
  • 完整病史记录：影像特征、症状、既往病史等
  • 自动生成专业健康管理报告
  • 支持 HTML 预览和 PDF 导出
🚀 使用 Vue 前端获得最佳体验

当前显示的是传统模式。要使用完整的多结节表单功能，请：

  1. 启动 Vue 前端：cd frontend/vue-app && npm run dev
  2. 访问：http://localhost:5173/nodule-form
