系统架构概览

实际运行架构

您的系统只使用B端单结节系统，通过一套数据模型 + 动态字段 + 动态模板选择的方式支持七类结节管理。

实际运行的系统：
┌─────────────────────────────────────────────────────────┐
│  B端系统 (BPatient + BHealthRecord + BReport)            │
│  - BPatient.nodule_type: 7种类型                         │
│  - BHealthRecord: 76个字段（包含所有器官）                │
│  - 根据nodule_type动态使用字段                            │
└─────────────────────────────────────────────────────────┘

未使用的系统（冗余代码）：

技术栈

  • 后端：Flask + PostgreSQL + SQLAlchemy
  • 前端：Vue 3 + Vite
  • LLM：OpenRouter API（deepseek-chat）
  • PDF生成：Playwright
  • 部署：独立部署（Flask运行在5000端口，Vue打包后集成到Flask）

BPatient 模型

class BPatient(db.Model):
    nodule_type = db.Column(db.String(50))  # 核心字段

    # - 'breast'（乳腺）
    # - 'lung'（肺部）
    # - 'thyroid'（甲状腺）
    # - 'breast_lung'（乳腺+肺）
    # - 'breast_thyroid'（乳腺+甲状腺）
    # - 'lung_thyroid'（肺+甲状腺）
    # - 'triple'（三结节）

class BHealthRecord(db.Model):
    # === 通用字段 ===
    age, height, weight, family_history...

完整业务流程

主流程图

1. 创建患者
   └─ B端管理师在 PatientsView.vue 创建患者
   └─ 选择 nodule_type（7选1）
   └─ POST /api/b/patients
   └─ 保存到 b_patients 表

2. 建立健康档案
   └─ 根据 patient.nodule_type 跳转到对应表单
   └─ 填写档案（76个字段中的子集）
   └─ POST /api/b/records
   └─ 保存到 b_health_records 表
   └─ 自动调用 LLM 生成 AI 西医评估（后台）

3. 生成报告草稿
   └─ POST /api/b/reports/generate
   ├─ 知识库匹配（match_knowledge_v2_by_source_type）
   ├─ 决策树处理（decision_tree.process）
   ├─ LLM 生成内容（3次API调用）：
   │   ├─ 影像学结论（generate_imaging_conclusion_with_llm）
   │   ├─ 综合分析结论（generate_comprehensive_conclusion_with_llm）
   │   └─ 分类建议（generate_recommendations_by_category）
   ├─ 根据 nodule_type 选择模板（report_manager.get_template_path）
   ├─ 提取字段（report_manager.extract_template_fields）
   ├─ 渲染 HTML
   └─ 保存到 b_reports 表（status='draft'）

4. 报告审核
   └─ B端管理师在 ReportViewView.vue 编辑报告
   └─ PUT /api/b/reports/:id/sections
   └─ 更新四个 section 字段
   └─ 重新渲染 HTML

5. 发布报告
   └─ PUT /api/b/reports/:id/publish
   └─ status: 'draft' → 'published'

6. 导出 PDF
   └─ GET /api/b/reports/:id/export_pdf
   └─ 使用 Playwright 生成 PDF
   └─ 返回下载

# backend/services/image_analysis_service.py（新建文件）
import requests
from config import Config

class ImageAnalysisService:
    """影像识别服务"""

    def __init__(self):
        self.api_key = Config.MEDICAL_AI_API_KEY
        self.api_url = Config.MEDICAL_AI_API_URL

    def analyze_ultrasound_report(self, image_path: str, nodule_type: str = 'breast') -> dict:
        """
        识别超声检查报告

        Args:
            image_path: 图片文件路径
            nodule_type: 结节类型（breast/lung/thyroid）

        Returns:
            识别结果字典
        """
        # 1. 读取图片
        with open(image_path, 'rb') as f:
            image_data = f.read()

        # 2. 调用第三方API（示例：阿里云医疗AI）
        response = requests.post(
            f'{self.api_url}/medical/ultrasound/analyze',
            headers={'Authorization': f'Bearer {self.api_key}'},
            files={'image': image_data},
            data={'report_type': nodule_type}
        )

        result = response.json()

        # 3. 解析结果
        parsed = {
            'birads_level': self._extract_birads(result),
            'nodule_size': self._extract_size(result),
            'nodule_location': self._extract_location(result),
            'boundary_features': self._extract_boundary(result),
            'internal_echo': self._extract_echo(result),
            'blood_flow_signal': self._extract_blood_flow(result),
            'confidence': result.get('confidence', 0.0)
        }

        return parsed

    def _extract_birads(self, result: dict) -> str:
        """提取BI-RADS分级"""
        # 正则匹配：BI-RADS 3、BI-RADS 4A等
        import re
        text = result.get('ocr_text', '')
        match = re.search(r'BI-?RADS[：:\s]*([0-6][ABC]?)', text, re.IGNORECASE)
        if match:
            return match.group(1)
        return None

    def _extract_size(self, result: dict) -> str:
        """提取结节大小"""
        # 正则匹配：约8.2mm、大小10×8mm等
        import re
        text = result.get('ocr_text', '')
        match = re.search(r'(\d+\.?\d*)\s*[x×]\s*(\d+\.?\d*)\s*mm', text)
        if match:
            return f'{match.group(1)}×{match.group(2)}mm'

        match = re.search(r'(\d+\.?\d*)\s*mm', text)
        if match:
            return f'{match.group(1)}mm'

        return None

            created_at=datetime.now()
        )
        db.session.add(record)

        # 3. 更新提交记录
        submission = MiniProgramFormSubmission.query.filter_by(
            patient_id=patient_id,
            completion_status='draft'
        ).first()

        if submission:
            submission.completion_status = 'completed'
            submission.submitted_at = datetime.now()

        if form_data.get('ultrasound_report_url'):

                form_data['ultrasound_report_url'],

def _calculate_completion_rate(form_data: dict) -> float:
    """计算表单完成度"""
    required_fields = ['name', 'age', 'gender', 'phone', 'nodule_type', 'family_history']
    optional_fields = ['symptoms', 'birads_level', 'ultrasound_report_url']

    completed_required = sum(1 for field in required_fields if form_data.get(field))
    completed_optional = sum(1 for field in optional_fields if form_data.get(field))

    # 必填项权重70%，选填项权重30%
    rate = (completed_required / len(required_fields)) * 0.7 + \
           (completed_optional / len(optional_fields)) * 0.3

    return round(rate, 2)

def assign_manager_to_patient(patient: CPatient) -> User:
    """分配管理师（简单轮询策略）"""
    # 获取所有活跃的管理师
    managers = User.query.filter_by(
        role='health_manager',
        is_active=True
    ).all()

    if not managers:
        return None

    # 统计每个管理师的患者数
    manager_loads = []
    for manager in managers:
        patient_count = BPatient.query.filter_by(manager_id=manager.id).count()
        manager_loads.append((manager, patient_count))

    # 选择患者数最少的管理师
    manager_loads.sort(key=lambda x: x[1])
    selected_manager = manager_loads[0][0]

    # 更新患者的分配
    b_patient = BPatient.query.filter_by(phone=patient.phone).first()
    if b_patient:
        b_patient.manager_id = selected_manager.id
        db.session.commit()

    return selected_manager
