# 清除缓存并重启服务器指南

## 问题现象
乳腺+甲状腺表单只显示乳腺部分，但代码中两个部分都存在

## 解决步骤

### 1. 停止开发服务器
按 `Ctrl + C` 停止当前运行的 `npm run dev`

### 2. 清除构建缓存
```bash
cd frontend/vue-app
rm -rf node_modules/.vite
rm -rf dist
```

### 3. 重新启动开发服务器
```bash
npm run dev
```

### 4. 清除浏览器缓存
**Chrome/Edge:**
1. 打开开发者工具 (F12)
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

**或者:**
- 按 `Ctrl + Shift + R` (Windows)
- 按 `Cmd + Shift + R` (Mac)

### 5. 重新访问页面
完全关闭浏览器标签页，重新打开应用

## 验证修复
1. 登录系统
2. 进入患者详情页
3. 点击"创建健康档案"
4. 选择"乳腺+甲状腺双结节"
5. 检查表单是否显示：
   - ✓ 基本信息
   - ✓ 乳腺结节数据（BI-RADS分级等）
   - ✓ 甲状腺结节数据（TI-RADS分级等）
   - ✓ 乳腺基础病史
   - ✓ 甲状腺基础病史
   - ✓ 其他风险因素
